9. ***唯一性定理***：要确定一个空间区域V中的静电场，我们需要以下四个条件：
	1. 文字表述：在V内存在唯一的解，**它在每个均匀区域内满足泊松方程**，在每个区域分界面上满足**边值关系**，并在V的边界S上满足给定的$phi$或者$(partial phi) / (partial n)$
	2. 在区域的每一个均匀的介质子区域$V_(i)$内满足泊松方程$nabla^2phi=-rho_(f)/epsilon_(i)$
	3. 在区域V中每两个介质子区域中的边界上知道$epsilon_(i)(partial phi_(i)) / (partial n)-epsilon_(j)(partial phi_(j)) / (partial n)=sigma_(f)$
	4. 在导体的时候，我们需要知道所有导体的电量$Q$或者$phi_(i)$
	5. 以及我们必须知道V的表面S上的$phi$或者$(partial phi_(i)) / (partial n)$的值
	6. 无导体纯介质的证明：
		1. 我们假设在一个区域中存在两个解：$phi'$ and $phi''$，设$phi=phi'-phi''$
		2. 由于两者都在一个区域那么我们可以得到$nabla^2 phi=0$
		3. 由于只有介质，在两个介质的边界上必然存在两个结果：$phi_(i)=phi_(j)$，$epsilon_(i)(partial phi_(i)) / (partial n)-epsilon_(j)(partial phi_(j)) / (partial n)=0$，
		4.  则在整个区域面S上，我们知道$phi_(S)=phi'|_(S)-phi''|_(S)=0$（前提条件）
		5. 从而我们考虑这样的积分$integral epsilon_(i) phi nabla phi d arrow(S)=integral epsilon_(i) nabla dot (phi nabla phi )d arrow(V)$，我们可以将它展开为$integral  (nabla phi)^2 d arrow(V)+ integral phi nabla^2phi d arrow(V)$。将上式代入，第二式为0。
		6. 对所有的i求和,$Sigma_(i)integral epsilon_(i) phi nabla phi d arrow(S)=Sigma_(i) integral epsilon_(i)  (nabla phi )^2 d arrow(V)$，我们得到我们知道$nabla phi_(i)=(partial phi_(i)) / (partial n) epsilon_(i)=(partial phi_(j)) / (partial n) epsilon_(j)$，且每两个积分之间的$d S$方向相反，从而只剩下边界的$phi_(S)$且由(4)，边界上$phi=0$，从而左式积分为0，则得到右式$Sigma_(i) integral epsilon_(i)(nabla phi)^2 d V=0$
		7. 由于被积函数必然大于0，从而我们得到$nabla phi=0$，从而$phi'-phi''="constant"$，相差常数对电场无任何影响，从而得到电场必然唯一
