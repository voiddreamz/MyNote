## 2. 最小作用量原理

2. 在朗道的这个体系，或者说现代的宏观物理体系，我们必然要假设**最小作用量原理（Hamilton's Principle / Principle of Least Action）**是一个公理。
3. 我们设$q(t)$是满足S取得最小值，从而我们得到的任意的函数$q+ delta q$去替代它都将使得S增大。其中的$delta q(t)$在$t_1$和$t_2$间隔都是小量。由于我们这个问题的前提是确定在$q(t_1)$和$q(t_2)$的值，从而我们知道$delta q(t_1)=delta q(t_2)=0$。
4. 那么从最小作用量原理表示开始：$delta S=delta  integral^(t_2)_(t_1)L=0$
5. 其变分后的形式：我们要将其变分，是求它的增量，那么这个时候我们可以考虑使用泰勒展开：$delta L=L(q,dot(q),t)+(partial L) / (partial q)delta q+(partial L) / (partial dot(q))delta dot(q))+dots-L(q,dot(q),t)=(partial L) / (partial q)delta q+(partial L) / (partial dot(q))delta dot(q)$。
6. 即是说最小作用量的微分形式为$$delta S= integral^(t_2)_(t_1)((partial L) / (partial q)delta q+(partial L) / (partial dot(q))delta dot(q))d t=0$$
7. 我们对它做小小的变换：我们将第二项的 $delta dot(q)$ 不满意，想要一个只有 $q$ 的方程，我们知道它等于 $d /(d t)delta q$，从而我们代入原式，从而得到
8. $delta S= integral^(t_2)_(t_1)((partial L) / (partial q) delta q d t+d/(d t)(partial L) / (partial dot(q)) delta q) =0$，那么我们将第二项运用分部积分（因为我们的 $d/(d t)((partial L) / (partial q)) delta q=d/(d t)((partial L) / (partial q)q)-(partial L) / (partial q)dot(q)$），得到了 $(partial L) / (partial dot(q))delta q|^(t_2)_(t_1)+ integral ((partial L) / (partial q)-d / (d t) ((partial L) / (partial dot(q))))delta q d t=0$
9. 由于我们知道 $delta q(t_1)=delta q(t_2)=0$，从而第一项必然等于0。由于第二项我们知道 $delta q$ 是随意的，其总和必然不等于0。
10. 则我们能得到**Lagrange's Equations**$$(partial L) / (partial q)-d / (d t) ((partial L) / (partial dot(q)))=0$$
11. 对于有$s$个自由度的系统，显然我们能得到$s$个方程：$(partial L) / (partial q_(i))-d / (d t) ((partial L) / (partial dot(q_(i))))=0$
12. 显然，我们对力学系统的拉格朗日函数乘以一个任意常数，不会改变运动微分方程。这似乎导致一种严重的不确定性：各个孤立力学系统的拉格朗日函数将乘以完全不同的任意常数！那么为了消除这个不确定性，只允许所有系统的拉格朗日函数都乘以同一个任意常数，而这归结为选择这个物理度量单位的任意性。
	1. **对上文解释**：首先，我们将Lagrange函数 $L -> C L$，我们将它代入Lagrange's Equations，自然不会变，结果仍然是0！从而我们得到：**拉格朗日函数的绝对数值不重要，重要的是它是否平衡**。
	2. 我们将所有的拉格朗日函数乘以一个常数，实际是对它换算了单位！那么粒子的运动会变吗？显然不会。
	3. 朗道说的"度量单位的任意自然性"，是指我们可以自由选择以下单位：
		*   能量单位（改变 $L$ 的整体倍数）
		*   时间单位（改变 $d t$ 的标度）
		*   长度单位（改变 $q$ 的标度）
		- 只要我们对所有物理系统统一改变，物理定律的数学形式保持不变。
	 4. 所以说朗道这里想说的一个非常重要的一点：**物理定律描述的是"关系"，而不是"数值"。**
13. 对于更加一般性的讨论
	1. 我们考虑两个拉格朗日函数 $L$ and $L'$。它们相差某个坐标和时间的函数 $f(q,t)$ 对时间的全导数：$L'=L+d/(d t)f(q,t)$
	2. 先等等，在数学运算之前，**我们为什么在物理上这样成立？我们恐怕要从 Lagrange 量的真实意义开始看起**
		1. 首先我们知道，最小作用量原理是在干嘛？证明 Lagrange 量这个记分板，来给我们的所有运动计分，计分的结果是最小的。
		2. 既然是计分的，那么我们只需要计算就行，所有的人的起始和结束都加上多少分，谁该最低谁还是最低。
		3. 那么我们现在来考虑为什么是它这样的一个式子能让它不影响结果？
		4. 首先，我们对 $d/(d t)f(q,t)$ 积分，相当于我们得到了它两个端点的值之差，即是说，它还是只影响了起点和终点。而我们在计算最终的 $delta S$ 的时候，我们的起点和终点都是它，所以无变化
		5. 那么我们自然是不能让这个函数和速度 $dot(q)$ 有关，因为我们的变分没有固定它的起点终点的速度的值。
		6. 简单来说，加时间的全导数是因为它在积分后将只剩下我们不考虑的起始和终点位置。我们不能有 $dot(q)$ 是因为我们的变分不能固定速度。
	3. 那么我们计算这两个拉格朗日函数对应的积分，即作用量：
		1. $$S' &= integral^(t_2)_(t_1)L(q,dot(q),t) d t+d/(d t)f(q,t)d t \
	&=S+f(q^((2)),t_2)	-f(q^((1)),t_1)$$
		2. 即是说，它们相差一个附加项。该附加项将在变分的时候消失，条件$delta S'=0$和$delta S=0$完全等价，因而运动微分方程也相同。
			1. 针对这个的推导？对于变分，我们有这样的前提：$q(t_1)$ $q(t_2)$均是已知量且不变。
			2. 从而我们对它变分，因为它不变，它的边界处变分就是0.
		3. 这意味着什么？**拉格朗日量不是唯一确定的**。物理上可观测的，只有运动方程，而运动方程对乘以常数和加上只包含$f(q,t)$的全微分的变换都不变。
## 3. Galileo 相对性原理
 1. 对于任意的一个参考系，它**都是非均匀且各向异性**的。即是说，如果某个物体与其他物体之间没有相互作用，它**在空间的不同位置和不同指向在力学意义上是不等价的**。
	 1. 均匀性：空间在所有位置的性质完全相同。即是说如果你把一个物体从位置 A 移到位置 B，它的力学行为完全一样。但是 在圆盘的中心：小球几乎直直地落下（因为中心旋转速度小），在圆盘的边缘：小球会明显被"甩出去"（科里奥利力效应明显）；而 在圆盘的边缘：小球会明显被"甩出去"（科里奥利力效应明显）
	2. 各向同性：空间在所有方向的性质完全相同。即是说如果你把一个物体旋转 90°，它的力学行为完全一样。将小球沿径向向外抛：小球会被科里奥利力偏向一边；沿径向向内抛：小球会被科里奥利力偏向另一边
	3. 从而对于这段内容的描述：我们确实知道如果我们就随便使用这样的混乱参考系，我们将难以对一个物体进行计算！
2. 从而我们是否存在这样的参考系，使得时间空间*相对我们要研究的物体*是均匀且各向同性的？有的！**惯性参考系**！
	1. 注意，这不是推导出来的，这是朗道用"对称性"来重新定义惯性参考系
		1. 首先我们如果要让它均匀（与位置无关）？那么$L$必然不显含$q$
		2. 接着我们要让它各向同性（与方向无关）？那么$L$必然只能依赖速率而非速度（或者说速度的平方）
		3. 最后我们要求时间均匀（与时间无关）？那么$L$必然不显含$t$
		4. 综上，$L=L(dot(q)^2)$，
		5. 那么我们将这样的一个式子代入Eula-Lagrange方程，$d/(d t) (partial L)/(partial dot(q)) - (partial L)/(partial q) = 0$。
		6. 由于第二项等于0，第一项中的$(partial L) / (partial dot(q))=2 dot(q) dot L^(')(dot(q)^2)="const"$
			1. 对于上式的推导过程，其实我们可以这样考虑：因为L是一个$v^2$的函数，所以它针对$v$的导数必然是一个以$v$的函数，但是由于它是常数，则$v$必然不能是一个变化的值，从而它必然是一个常数。
		7. 从而我们知道了，在我们这样的假设下，我们必然能得到我们想要的物体匀速直线（或静止）运动。
	2. 那么针对这样的惯性参考系，我们知道任何的一个质点速度的大小和方向均不改变，我们称之为**惯性定律**
3. 在无论多少个惯性参考系中，它们互作匀速直线运动。在这些参考系中的空间和时间的性质都是相同的。我们称之为**Galileo相对性原理**
	1. 关于这个，我们可以更加翻译为对于两个惯性参考系中的运动方程必须相同的，即是说 Lagrange 方程必然相同。
4. 在讲下一个主题之前，我们先有一个前提，**绝对时间假设**：即是说，我们认为*在不同的参考系中的时间是相同的*。
5. Galileo变换：
	1. 我们先假设存在两个不同的参考系$K$和$K'$，其中$K‘$相对$K$以速度$arrow(V)$运动。同一个质点相对这两个参考系坐标$arrow(r)$和$arrow(r')$满足关系式：$$arrow(r)=arrow(r')+arrow(V)t$$
	2. 同时认为两个参考系中的时间是相同的：$$t=t'$$
	3. 我们将上式称作**Galileo变换**。
	4. 从而我们可以将Galileo相对性原理表述为：力学方程在伽利略变换下具有不变性。
## 4. 自由质点的拉格朗日函数
1. 首先我们研究一个最简单的例子：质点相对惯性参考系的自由运动。
	1. 在这样的情况下，我们有上文知道 $L$ 只和 $v^2$ 有关。那么我们利用 Galileo变换来讨论这个问题。如果惯性参考系 $K'$ 以无穷小的速度 $epsilon$ 相对另一个惯性参考系 $K$ 运动，我们有 $v'=v+epsilon$，这意味对于 Lagrange 函数 $L(v^2)$，经过变换后得到了 $L'=L(v^2)=L(v^2+2 arrow(v)dot arrow(epsilon)+epsilon^2)$ ，我们将它展开为 $epsilon$ 的幂函数（注意，我们的 $L'$ 也是一个只和 $v^2$ 有关的函数）并忽略一阶以上的无穷小量
		1. 注意，$L(u + delta u) approx L(u) + (d L)/(d u) dot delta u$ ，即是说 $L(u + delta u)-L(u)  = (d L)/(d u) dot delta u+dots$
		2. 从而针对原式，我们得到 $L(v^(' 2)) approx L(v^2) + (d L)/(d(v^2)) dot.op (2 arrow(v) dot.op arrow(epsilon) + epsilon^2)$
		3. 同时我们要求忽略它的一阶以上的无穷小量，从而我们只剩下了 $L(v^(' 2)) = L(v^2) + (d L)/(d(v^2)) dot.op 2 arrow(v) dot.op arrow(epsilon)$
		4. **物理思维** 为什么我们这里舍弃了 二阶及以上的高阶无穷小？我们需要的是 $delta L=L'-L$，即是说第一点，并且是 $lim_(epsilon arrow.r 0)(delta L)/epsilon$，那么二阶以上的内容将在趋向于 0 的时候消失，从而实现了表面上说的*舍弃了二阶及以上的高阶无穷小* 。
		5. 又由于我们之前得到了由于 Galileo 相对性原理，在两个惯性参考系中，我们的运动方程必须相同，这时我们只允许它们相差一个 $f(q,t)$ 的全导数。
		6. 首先对于 $arrow(v) dot arrow(epsilon)$ ，它就是一个 t 的全导数，因为 $arrow(v)dot arrow(epsilon)=d/(d t)dot epsilon$，其中因为 $epsilon$ 它是一个常数，从而 $arrow(v)dot arrow(epsilon)=d/(d t)dot epsilon=(d(arrow(v) dot epsilon))/(d t)$ 
		7. 那么因为后面的部分 $arrow(v) dot arrow(epsilon)$ 是一个关于 t 的全导数，那么要让整体是一个全导数，前面的系数 $2 (partial L) / (partial v^2)$ 也要与速度无关。我们设 $2(partial L) / (partial v^2) =m$，其中的 $m$ 是一个常数。
		8. 那么我们 得到了 $$L(v^2)=1/2 m v^2$$
		9. 由拉格朗日函数在速度无穷小变换下满足 Galileo 变换，在参考系 $K'$ 以有限速度 $arrow(V)$ 相对 $K$ 运动的情况下，Lagrange 函数仍然满足该原理，
			1. 即是说，$L'=m/2((v')^2)=m/2 v^2+m/2 arrow(v)dot arrow(V)+m/2 V^2$，或者说 $L(v^(' 2)) = L(v^2) + (d L)/(d(v^2)) dot.op (2 arrow(v) dot.op arrow(V) + V^2)$，而第二项可以略去。
	2. 那么我们将这里面出现的物理量 $m$ 称为**质量**。
	3. 由于 Lagrange 函数的可加性，对于无相互作用的质点组成的自由质点系，有 $$L=sum_(a)(m_(a)v^2_(a))/2$$
		1. 这里我们必须指出，给出质量的定义才有实际物理意义。我们乘上一个常数就相当于改变了质量的单位。从而不同质点的质量的比例关系确实具有实际的物理意义，它将不改变也不能改变。
	4. 根据最小作用量原理，质点从空间点 1 到点 2 的真实运动，使得：$S= integral^(2)_(1)(m v^2)/2$ 取最小值。从而我们要求 $m>0$. 假设质量是负的，对于质点快速离开点 1 再快速接近点 2 的轨迹，作用量可以取得绝对值任意大的负值，但是绝对不能取最小值。   
		1. 为什么能有这样的结论？当我取这个路径的时候，我们能得到：$v_1$ 和 $v_2$ 将可以非常大（因为是快速离开 1 和接近 2）！
		2. 在这种情况下，如果 $m<0$，我们必然能得到 $L = 1/2m v^2 < 0$，作用量在这样的情况下，也会小于零。
		3. 问题在于什么？我们可以让这个远离的速度 $v_1$ 和 接近的速度 $v_2$ 趋向无穷大，从而作用量 $S$ 也可以取得绝对值无穷大！在这样的情况下，我们必然不可能有最小值（总能更小！即是说最小作用量原理失效！
		4. 从而我们反证得到 $m>0$
	5. 注意到 $v^2=(d l^2)/(d t^2)$。从而在不同的坐标系中，我们只需要求出 $d l^2$ 的表示就可以表示出它的 Lagrange 函数。
		1. 例如在笛卡尔坐标系中，$d l^2=d x^2+d y^2+d z^2$，从而它的 Lagrange 函数表示为 $L= m/2(dot(x)^2+dot(y)^2+dot(z)^2)$
## 5. 质点系的 Lagrange 函数
1. **封闭质点系**：质点之间有相互作用，但是不受外部的物体的作用。
2. 在这样的质点系下我们要描述质点之间的相互作用，可以在自由质点系的 Lagrange 函数（1.4.3）上增加某一个坐标相关的一个函数（之所以只有坐标有关而和时间无关，是因为相互作用只和位置有关，在一个封闭的质点系之间，我们的时间是均匀的，没有外界干扰） ，记为 $U(q_1,q_2,dots)$，即是说，$$L=sum_(a)(m_(a)v_(a)^2)/2-U(r_1,r_2,dots)$$
3. 那么我们称 $U$ 为质点系的势能，而 $T=sum_(a)(m_(a)v_(a)^2)/2$ 称之为动能。
4. 由于势能只依赖于所有质点同一时刻的位置，这意味着其中任意质点的位置的变化将直接影响到所有其它质点，而且相互作用是瞬时的。（这个性质在经典力学中是必然的，它紧密联系着经典力学的基本前提，即绝对时间的假设和伽利略相对原理）。
	1. 如果说它不是瞬时的，而是以一个有限的速度传递
	2. 由于时间的绝对性，意味着速度相加适用于所有的现象
		1. 关于这一点，我们从速度的定义，$v=(d x)/(d t)$。再加上洛仑兹变换（位移及时间的不变性），我们能得到 $v'=v-V$
	3. 从而在有相对运动的不同参考系中传递速度将会不相同。从而相互作用的物体在运动规律不同惯性系下将不同，违反了 Galileo 相对性原理。（但是事实是实验证明了相互作用有一个最大速度，详情参见[[场论]]） 
5. 在之前我们只提到了惯性系中时间的均匀。但是实际通过上文的 1.5.2 的公式，我们得到了：时间不仅是均匀的，而且是各向同性的。事实上，我们将 $t$ 用 $-t$ 来代替，完全不会改变 Lagrange 函数，从而我们得到了这样的一个结论：**遵循经典力学的所有运动是可逆的**
6. 由 Lagrange 方程 $d/(d t)((partial L) / (partial arrow(v)))=(partial L) / (partial arrow(r))$，我们将 5.2 式（$L=T-U$）代入。由于动能 $T$ 只与速度有关，势能 $U$ 只与坐标有关，我们得到了这样的一个结果：$$m (d arrow(v))/(d t)=-(partial U) / (partial arrow(r))$$ 我们称之为**牛顿方程**。
7. **关于“力”的定义**： 我们定义 $arrow(F)=-(partial U) / (partial arrow(r))$ ，称之为**作用在质点上的力**。
	* **为什么要这样定义力？** 在传统的牛顿力学中，力是一个自生自发的前提，但在分析力学中，根本没有力的位置，只有势能 $U$ 和动能 $T$。
	* 真正的物理逻辑是：自然界的空间中存在某种势能分布，势能在空间上的变化（即负梯度 $-(partial U) / (partial arrow(r))$）打破了空间的均匀性。这种不均匀性“迫使”物体的运动状态发生改变（产生加速度 $m (d arrow(v))/(d t)$）。
	* 因为这种势能的差异**改变了物体的运动**，为了在宏观上直观地描述这种“促使运动改变的原因”，我们用这个势能的空间变化率**定义出了“力”**的概念。这就是为什么力 $arrow(F)$ 只依赖于坐标 $r$，而与速度无关。
	2. 同时，由上述推导，我们明确知道了质点的加速度仅由它当前的空间坐标决定。
	3. 我们之前的 1.2.13 的内容，我们知道我们可以特殊地取势能增加一个常数。最自然的，我们可以选择当无线增大质点间的距离的时候，他们的势能将无限趋近于 0。
8. 但是不是所有的时候我们的 Lagrange 函数都要用笛卡尔坐标系。我们是时候去使用广义坐标来表示 Lagrange 函数了
	1. 首先我们能轻易得到这样的表达式：$x_(a)=f_(a)(q_1,q_2,q_3,dots)$，$dot(x)_(a)=sum_(k)(partial f_(a)) / (partial q_(k))dot(q)_(k)$
	2. 我们将它们代入笛卡尔坐标系的动能表达式 $L=1/2 sum_(a)m_(a)(dot(x)_(a)^2+dot(y)_(a)^2+dot(z)_(a)^2)-U$ 
	3. 将得到最终的结果：$$L = 1/2 sum_(i,k) a_(i k)(q) dot(q)_i dot(q)_k - U(q)$$
9. **【物理直觉插入：理解广义动能与质量矩阵】**
	* **这坨复杂的求和到底是什么？** 前面的 $1/2 sum_(i,k) a_(i k)(q) dot(q)_i dot(q)_k$ 物理本质上就是系统的**总动能 $T$**。在广义坐标下，速度变成了广义速度 $dot(q)$（可能是线速度，也可能是角速度、伸缩率等），所以动能自然变成了不同广义速度项交叉相乘的组合。
	* **$a_(i k)(q)$ 是什么？** 它是**广义质量矩阵（或叫惯性张量）**。
    *  **为什么是矩阵？** 因为复杂系统存在**运动的耦合**。对角线元素代表单纯沿着方向运动的"有效质量"；非对角线元素代表运动间的干涉（例如机械臂转动大臂时，小臂也会由于结构限制感受到额外的惯性力）。
	* **为什么它依赖于坐标 $q$？** 这是理论物理最精彩的直觉！举个例子：极坐标下的质点动能为 $T = 1/2 m dot(r)^2 + 1/2 (m r^2) dot(theta)^2$。沿着角度 $theta$ 方向的"有效质量"是转动惯量 $m r^2$，它显然依赖于质点当前离中心的距离 $r$！这在物理上意味着：**物体在不同的姿态（坐标 $q$）下，朝着不同的方向运动时，感受到的“沉重感”（惯性大小）是完全不同的。**
	* **理论物理的终极视角**：在这个多自由度系统里，物体实际上是在一个弯曲的“位形空间（Configuration Space）”里运动。而 $a_(i k)(q)$ 就是这个抽象空间的**度规张量（Metric Tensor）**。它定义了空间的弯曲程度和距离（爱因斯坦后来正是借用了这套数学语言，将质量矩阵替换为时空度规 $g_(mu nu)$，创立了广义相对论）。
10. 非封闭质点系（外场）的 Lagrange 函数
	1. **上帝视角的起点：** 当我们研究一个受“外场”作用的系统 A 时，真正的物理场景是：宇宙中存在系统 A 和一个极其巨大的系统 B，它们共同构成了一个**封闭系统 A+B**。因为是封闭系统，整体的拉格朗日量不显含时间 $t$：
   $$L = T_A(dot(q)_A) + T_B(dot(q)_B) - U(q_A, q_B)$$
	2. **定义什么是外场：** 由于系统 B 极其庞大（例如地球、巨大的电容器），微小的系统 A 对系统 B 的反作用力完全可以忽略。这意味着，系统 B 将完全按照自己固有的规律运动，**它的轨迹 $q_B(t)$ 是已知且确定的**，不受 A 的影响。
	3. 我们将已知的 $q_B(t)$ 代入总的拉格朗日量：
		1. 动能 $T_B(dot(q)_B)$ 变成了一个纯时间的已知函数 $f(t)$。在拉格朗日力学中，纯时间函数可以写成全导数 $d/(d t) (integral f(t) d t)$，根据之前推导的结论，**全导数项对运动方程无影响，可以直接舍弃！**
		2. 势能 $U(q_A, q_B)$ 由于代入了 $q_B(t)$，变成了一个**显含时间**的函数 $U(q_A, t)$。
	4. **最终的系统 A 的拉格朗日函数：** 舍弃无关项后，我们得到了受外场作用的小系统 A 的独立拉格朗日量：
   $$L_A = T_A(dot(q)_A) - U(q_A, t)$$
	5. **【物理直觉插入：外场的本质与时间的破缺】**
		1. **外场并不是什么神秘的魔法。** 它是“宇宙中被我们忽略掉的那一部分巨大物体（系统 B）的运动残影”。
		2. **为什么坐标 $q(t)$ 本身是时间的函数，但封闭系统的 $L$ 却“不显含”时间 $t$？** 
     - 这里的核心物理图像是：**物理规律是否随“日期（Date）”改变？** 想象两个相互吸引的星球在真空中运动。它们的坐标 $arrow(r)_1(t)$ 和 $arrow(r)_2(t)$ 确实随着时间流逝在不断变化。但是！它们的势能公式 $U = -G m_1 m_2 / |arrow(r)_1 - arrow(r)_2|$ 这个“规律”本身，**只在乎它们当前的空间相对位置（State），根本不在乎现在是几点（Date）**。
     - 数学上的检验方法：我们把这两个星球**死死钉在相距 1 万公里的位置上不动（固定状态 $q$ 和 $dot(q)$）**，然后让墙上的时钟 $t$ 滴答滴答走。你会发现，无论时间怎么流逝，它们之间的势能永远是那个固定的数值！这说明势能函数里没有挂着一个独立的时间变量 $t$。这叫**隐含（Implicit）**。数学上体现为 $L(q, dot(q))$，偏导数 $(partial L)/(partial t) = 0$。这反映了：**时间是均匀的（物理规律不随时间改变），因此能量守恒**。
   * **为什么外场势能却“显含”时间 $t$？** 
     - **显含 $t$：** 现在想象把一个电子放进一台核磁共振（MRI）机器里。这台机器（外部巨大系统 B）由外部的医生操控，它的磁场强度按照预设的程序 $B(t) = sin(omega t)$ 在疯狂振荡。
     - 此时，即使你用镊子把这个电子**死死钉在机器正中心不动（固定坐标 $q_0$）**，让时钟 $t$ 继续走。你会惊恐地发现：这个电子感受到的势能，竟然在随着时间疯狂变化！**昨天它停在这里的势能，和今天停在这里的势能不一样了！** 
     - 因为那个外部巨大系统 B 的运动规律，变成了一个强行闯入的“外部时钟”，赤裸裸地写在了势能公式表面：$U(q, t)$。这就叫**显含（Explicit）**。数学上体现为 $(partial L)/(partial t) eq.not 0$。
   * **对称性被打破的代价：** 在封闭系统中时间是均匀的（能量守恒）。但在外场中，由于 $U$ 显含时间 $t$，意味着外部的巨大系统 B（MRI机器）在源源不断地给你的系统 A 注入或抽走能量。对于被孤立出来的系统 A 而言，**能量不再守恒**！这就是外场物理图像的理论渊源。
