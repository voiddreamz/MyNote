# 第一章：相对性原理
## 1. 相互作用速度
1.  实验表明，我们的**相互作用并不是瞬时的**。相互作用的一个物体改变，需要一定时间才能在别的物体上显现效果。我们将这段时间和之间的距离相除，我们就能得到**相互作用的传播速度**
2. 一个物体向另一个物体相互作用，我们一般称作信号，从而我们将这个速度称作**信号速度**。
3. 由相对性原理，信号的传播速度在所有的惯性参考系中都是一样的。那么我们将必须要求它的速度是一个普适常数，我们之后将会需要证明它是光速。
4. 我们将相对性原理和传播速度有限性结合起来，我们将能得到 **爱因斯坦相对性原理**。
5. 它的正确是实验中证明的。对于光如果说它是粒子，那么可以简单地按照 Galileo 变换来计算出它的速度。但是它的结果和声音一样，无论什么相对速度来移动，都是一个常数。
6. 并且由于时间并非是绝对的，从而我们将需要区分“同时”这个概念。两个事件在一个坐标系内同时，在另一个内将完全不同，我们可以简单地用这个图来看  ![[sametime.png]]
	7. 我们在 $K'$ 参考系上，我们从 A 点向 BC 两点同时发出信号，同时有以 $arrow(V)$ 相对于 $K'$ 参考系向左移动的 $K$ 参考系。在 $K'$ 中自然是同时到达 $B C$ 的，但是在 $K$ 中，信号的传播速度仍然是 $c$，我们从 $K$ 参考系（视作静止的），我们将发现 ABC 都是往右走的，那么 $B$ 自然是在靠近源，而 $C$ 在远离源。那么自然是信号更快传到了 $B$ 而传到 $C$ 更久了。

## 2. 间隔
1. 我们现在用数学的形式来表述光速不变定理。
2. 我们首先假设在 $t_1$ 的时候送出一个光信号，在 $t_2$ 的时候接受到了这个信号。那么这两个 Event 的空间上的间距我们可以用 $[(x_2-x_1)^2+(y_2-y_1)^2+(z_2-z_1)^2]^(1/2)$。同样的，我们可以用 $delta x=v delta t$ 来表示，即是说 $c(t_2-t_1)$。由于它们相等，我们可以得到 $c^2(t_2-t_1)^2-(x_2-x_1)^2-(y_2-y_1)^2-(z_2-z_1)^2=0$ 同样的，我们在另一个参考系中也能得到完全类似的结论。
3. 但是我们上文提到的 Event 是基于光速运动下推导的。在现实中大部分的运动都不是光速，从而右式将不等于零。从而我们将定义这样的一个物理量
4. $$s_(1 2=[c^2(t_2-t_1)^2-(x_2-x_1)^2-(y_2-y_1)^2-(z_2-z_1)^2])$$
5. 我们称之为两个 Event 的**间隔**（为什么我们这个式子和上文的推导的符号相反，因为我们大部分的运动无法达到光速，我们这样规定能让 $s>0$，我们将这个称作 $+ ---$ 度规约定，和广义相对论的反过来的度规区别）。或者在两个极度接近的 Event 中上式化作 $d s^2=c^2 d t^2 -d x^2 -d y^2-d z^2$
6. 我们由光速不变，自然能得到在任何一个参考系中我们都能得到 $d s=0,  d s'=0$（你或许会好奇，我们之前不是做过实验吗，在两个参考系下的两个同时事件将不再同时。这说明在不同坐标系下的不同的距离将会影响不同的时间，刚好证明了它是对的）。同时，由于空间和时间都是均匀的，我们能得到 $s^2$ 是线性的，那么自然对于不同的两个参考系，他们的间隔满足 $s^2=a s'^2$. 同时由于时间和空间都各向同性，我们能得到 $a=(V)$，且只取决于 $V$ 的大小而非方向，即是说我们可以得到 $a(-v)=a(v)$。同时我们可以由我们的时间反演可以得到同样的结论（因为 $d v= (d r)/(d t)$）。我们现在假设存在三个参考系，我们将能得到 $s^2_(1)=a(v_(12)) s^2_2, s^2_2=a(v_(23)) s^2_3, s^2_1=a(v_(13))s^2_3$，我们将第二式带入第一式，我们将能得到 $s^2_1=a(v_(12))dot a(v_(23)) s^2_3=a(v_(12))s^2_3$. 那么我们自然能得到 $a(v_(12))dot a(v_(23))=a(v_(13))$. 我们说它之和大小有关，但是速度必然是由两个矢量相加的，即是说我们得到的必然是存在角度有关的内容。但是左侧完全没有角度，即是说它如果真的和速度有关就错误了。那么它必然是一个常数。我们由同一个式子，我们能得到 $c dot c=c$，那么 $c=1$，即是说 $a=1$，即是说 $s^2=s'^2$
	1. 那么我们能得到一个这样的一个由光速不变得到的 $s=0=s'$ 推广出来的结论：**两个 Event 的间隔在任意一个参考系中都不变。**
	2. 其实它是由光速不变推导出来的，那么如果我们承认它，我们能完全由它来推导出我们的光速不变定理，即是说它也是**光速不变的数学表示**
		1. 那么我们现在手上有一个类似最小作用量原理的宇宙公理：$s=s'$
		2. 我们先假设在站台上以光速发出一个信号，那么在某一个点接收到了它，它们之间的间隔自然满足 $s=0$，那么在站台外经过的一辆车以速度 $V$ 经过站台，它的 $s'=s=0$. 我们也能得到 $delta r'^2=c^2 delta t'^2$. 我们将它给开放，除以 $t'$，我们将能得到 $(delta r)/(t')=c$。左边等于什么？就是我们测得的速度，它等于 $c$，即是说我们能得到在我们火车（另一个参考系）上，无论这个参考系的相对速度是多少，我们都能得到这个 Event 的速度。
7. 既然我们有了*间隔*的定义，我们现在开始研究它。那么我们先考虑一个情况：是否存在一个 $K'$ 使得本来在 $K$ 中的两个 Event 在同一点发生？
8. 我们自然有 $c^2delta t^2-delta l^2=c^2delta t'^2-delta l'^2$。同时，由于在 $K'$ 中我们让两个 Event 发生在一个点上，那么 $delta l'^2=0$，即是说 $$s^2=c^2 delta t^2-delta l^2=c^2 delta t'^2>0$$
	1. 当然，上式自然只能在当 $s$ 是实数的时候满足。则当两个 Event 的间隔为实数的情况才能满足它的平方大于零。
	2. 由于这样的间隔由时间决定，像时间一样，我们称这样的间隔为**类时间隔**
9. 同样的，如果我们的两个 Event 在同一个时间发生，那么同样，我们有 $$s^2=-delta l^2<0$$我们称之为类空间隔。但是由于间隔的平方是一个负数，我们需要我们的间隔是虚数才能找到我们的参考系。

## 3. 固有时
1. 我们现在考虑以一个钟的自己作为一个惯性参考系，我们将直接观察它的时间变化，而非通过观察一个运动在 $A$ 点的时间，和终点 $B$ 的时间，相减得到我们的时间。
2. 那么我们知道，在不同的坐标系中，我们的间隔是不变的。我们现在直接考虑一个随意取的坐标系和以它自身运动的惯性坐标系。在惯性坐标系下，我们的 $d l'=0$，即是说满足 $s^2=c^2d t'^2=c^2 d t^2-d x^2-d y^2-d z ^2$，那么我们将有 $d t'=d t sqrt(1-(d x^2+d y ^2+d z^2)/(d t^2 c^2))$，由于我们的 $(d x^2+d y ^2+d z^2)/(d t^2)=v^2$，那么我们将有 $$d t'=d t sqrt(1-v^2/c^2)$$ 我们将它积分，我们有 $$t_2'-t_1'= integral^(t_2)_(t_1)d t sqrt(1-v^2/c^2)$$ 我们称之为**固有时**。并且我们容易发现，我们的固有时必然比任意一个静止的坐标系相对的时间间隔小，或者说运动的钟的走得慢。
3. 但是这里我们如果从我们的时钟的参考系去看我们的外部的静止参考系，我们会发现我们的外部的时钟也是慢于我们的。不对吧，我们外部看我们的时钟是慢的，我们从时钟去看外部应该是相反的才对吧。

## Lorentz Transformation
1. 我们现在研究一个参考系，以 $V$ 的速度相对我们的原参考系运动，我们由我们的间隔不变，即是说 $d s=d s'$，我们考虑这个相对的运动发生在 $x$ 分量上，即是说 $d s^2= c^2 t^2-x^2=c^2 t' ^2-x' ^2=d s' ^2$
2. 那么我们现在要考虑 $T^2 -X^2=T' ^2- X'^2$，我们假设变化是线性的，即是说 $T=a T'+b X', X=c X'+d T'$，我们将它代入左侧，我们有 $a ^2T' ^2+ b^2 X' ^2- (d ^2 T' ^2+ c^2 X' ^2)= T^2- X^2$ 。我们两端系数必然相等，即是说我们有 $a^2-d^2=1,b^2-c^2=1,a c-b d=0$，那么我们只能想到双曲函数满足这样的性质（我们也可以在这里联想到高斯角，即是说三维的满足三角函数的关系）。那么我们将 $cosh Phi=a,b= sinh Phi,c = cosh Phi,d=  sinh Phi$ 代入，我们将有我们的 $T= cosh Phi T'+ sinh Phi X',X= cosh Phi X'+ sinh Phi T'$ .  那么我们现在考虑我们的运动的参考系的原点的运动，即是说我们的 $X'=0$，那么我们将有 $T= cosh Phi T', X= sinh Phi T'$.  我们将两式相除，我们有 $x/(c t)= tanh Phi$，其中我们注意到 $x/t=V$，则我们有 $V/c= tanh Phi$。我们将这个式子先表示 $cosh Phi= sinh Phi/(V/c)$ ，代入到我们的 $cosh ^2 Phi- sinh ^2 Phi=1$，后续忽略，我们将有我们 $cosh Phi=1/sqrt(1 -(V/c)^2)$, $sinh Phi=(V/c)/(sqrt(1-(V/c)^2))$，我们将得到我们 **Lorentz Transformation** 的最终表达式：$$cases(x= (x' + V t')/(sqrt(1-(V/c)^2)),t=(t'+ V/c^2 x')/(sqrt(1-(V/c)^2)))$$

4. 同样的，我们也可以由这样的式子来计算速度的变换。同时我们仍然考虑 $y,z$ 方向上不变。我们将前面的式子写作微分形式，即是说 $$cases(d x= (d x' + V d t')/(sqrt(1-(V/c)^2)),d t=(d t'+ V/c^2 d x')/(sqrt(1-(V/c)^2)),d y= d y',d z=d z')$$ 我们将它相除，用 $arrow(u)$ 来表示我们的其中的速度，我们将有**速度的变换** $$cases(arrow(v_(x))=(v_(x)'+V)/(1+V/c^2 u_(x)'),arrow(v_(y))=(v_(y) 'sqrt(1-(V/c)^2))/(1+V/c^2 u_(x)'),arrow(v_(z))=(v_(z) 'sqrt(1-(V/c)^2))/(1+V/c^2 u_(x)'))$$
5. 如果我们考虑**速度远小于光速**的情况，我们利用近似 $(1+x)^(-1)=(1-x)$ 以及我们的 $sqrt(1-x)=1+1/2 x$，我们将它带入我们之前的三个式子，我们可以将其简化，在这里不再写出，可自行推导。
