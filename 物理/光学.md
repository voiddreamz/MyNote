## 1. 波的干涉引论
1. 波的干涉产生原理：当我们研究两列波叠加的光强的时候
	1. 由于我们的光强等于 $I=n E^2$，我们研究只考虑关系，那么我们可以忽略系数，即是说 $I$ 正比于 $E^2=|U|^2=tilde(U) dot tilde(U)^*$
	2. 当我们的两列波叠加时，由于电磁场满足叠加原理，我们的光强也满足 $tilde(U)=A_1e^(i(arrow(k) dot arrow(r)-omega t+phi_0))+A_2e^(i(arrow(k) dot arrow(r)-omega t+phi_0))$ 		1. 我们将得到 $I=A_1^2+A_2^2+A_1A_2(e^(i(arrow(k)dot (arrow(r_1)- arrow(r_2))))+e^(-i arrow(k)(arrow(r_1)-arrow(r_2))))$
	3. 那么我们将它转化为实数形式，将得到 $$I=A_1^2+A_2^2+2A_1A_2 cos delta$$ 其中 $$delta=arrow(k) dot (arrow(r_1)-arrow(r_2))$$如果我们使用 $I_1,I_2$ 表示两束光的光强，我们将得到最终的光强是 $$I=I_1+I_2+2sqrt(I_1I_2) cos delta$$
	4. 即是说它不满足叠加原理。为什么它是电磁场却不行？因为**它不是先平方再叠加，而是先加再平方**
	5. 物理意义：这个 $cos( Delta phi)$ 项反映了两个波之间的相位关系对光强的调制作用。当 $Delta phi = 2n pi$（n为整数）时，$cos( Delta phi) = 1$，光强最大，产生相长干涉；当 $Delta phi = (2n+1) pi$ 时，$cos( Delta phi) = -1$，光强最小，产生相消干涉。
2. 相干叠加的条件
	+ 完全相干叠加有两个必要条件：
		+ 两列波频率必须相同（因为我们假设了它们的 $arrow(k)$ 相等才得出这样的结果）
		+ 要获得稳定的干涉场，必须保证场点有稳定的相位差（如果相位差真的不稳定，那么它们将分别为 $50 percent$ 的概率，那么结果将完全相互抵消，没有现象）
3. 相位差 $delta$
	1. 相位差的来源：
		1. 由于 $delta=arrow(k) dot (arrow(r_1)-arrow(r_2)$)
		2. 光程差：两个波传播的几何路径长度不同，$delta =  (2 pi) /( lambda) Delta L$，其中 $Delta L$ 是光程差，$lambda$ 是波长；(因为 $k=(2 pi)/lambda$)
			1. 如果其中的 $Delta L=m lambda,m in bold(Z)$ ，那么就等价于 $delta =2 m pi$ 。即 $delta =1$
			2. 如果 $Delta L=(m+1/2) lambda$，即 $delta =-1$
		 3. 初始相位差：两个波源本身可能有不同的初始相位，$Delta phi =  phi_1 -  phi_2$。
	2. 光程差与相位差的关系：光程差是指光在不同介质中传播时，由于折射率不同而导致的等效几何程差。$Delta L = n_1 L_1 - n_2 L_2$，其中$n$是折射率，$L$是几何路程。相位差$Delta phi =  (2 pi)/ ( lambda_0)n Delta L$，其中$lambda_0$是真空中的波长。

## 2. 杨氏双缝干涉
 1. 在中心点附近，我们的光程差只由 D 之间的的光程差导致，我们在傍轴附近，我们知道 $Delta L=r_2-r_1=sqrt(D^2+(x+d/2)^2)-sqrt(D^2+(x-d/2)^2)$
 2. 一些数学近似之后，我们就能得到 $$Delta L= (d x)/D$$
 3. 那么在干涉极大处，我们能得到 $Delta L=m lambda$，即是说 $x=m (lambda D)/d$ ，我们一般把它记作 $m$ 级条纹。
 4. 同理，在干涉极小处，我们能得到 $x=(m+1/2)(lambda D)/d$
 5. 从而两个明条纹或暗条纹之间的**条纹公式**：$$Delta x=(lambda D)/d$$
 6. 我们记得之前的光强的公式为 $I=I_1+I_2+2sqrt(I_1I_2) cos delta$，而我们的实验是一束光，则 $I_1=I_2$，代入后我们能得到 $I=2I+2I cos delta=2I(1+ cos delta)=2I(2  cos ^2 delta/2)$，由于我们知道 $delta =(2 pi)/lambda dot Delta L$，那么我们能得到 $$I=4 I_0 cos ^2 (pi d)/(lambda D)x$$ 即是说我们的光强和 $x$ 呈现余弦的变化形式。

## 空间相干性
1. 考虑现实中，我们必然是不存在一个真正的点光源。必然在纵向上有长度（横向的我们不用考虑，它只是增加条纹宽度罢了）

5. 那么我们将发现，在不同的地方，我们的亮暗纹将有不同的对比度我们将建立这样的一个物理量：**衬比度**
	1. $V = frac(I_(max) - I_(min),I_(max) + I_(min))$，其中 $I_(max)$ 是最大光强，$I_(min)$ 是最小光强。衬比度描述了干涉条纹的清晰程度，$V in [0,1]$。
	2. 为什么我们要用这样的式子而不是直接使用 $I_("min")/I_("max")$ 呢？因为我们一般想要让它满足这样的范围 $0<= gamma<=1$，因为当它为 0 时表示没有条纹了，最大的时候我们用 1 更加好用或'者写作 $100 percent$
	3. 衬比度的物理意义：衬比度反映了干涉的相干程度。$V=1$ 表示完全相干，干涉条纹最清晰；$V=0$ 表示完全不相干，无干涉条纹。衬比度越大，干涉条纹越清晰，测量精度越高。
	4. 衬比度与光强的关系：对于两束光干涉，$I_(max) = I_1 + I_2 + 2sqrt(I_1 I_2)$，$I_(min) = I_1 + I_2 - 2sqrt(I_1 I_2)$，因此衬比度 $V = frac(2sqrt(I_1 I_2),I_1 + I_2) = frac(2sqrt(I_1/I_2),1 + I_1/I_2)$。当 $I_1 = I_2$ 时，$V=1$，衬比度最大。
	5. 影响衬比度的因素：
		1. 光源的时间相干性：光源的单色性越好，相干长度越长，衬比度越高；
		2. 光源的空间相干性：光源的尺寸越小，空间相干性越好，衬比度越高；
		3. **光束的强度比：两束光强度越接近，衬比度越高。**
	6. 衬比度的应用：干涉测量中，衬比度是衡量测量精度的重要指标。在迈克尔逊干涉仪、激光干涉仪等精密测量仪器中，需要保持高衬比度以确保测量精度。
	7.  6.  **借用衬比度，我们可以进一步改写双光束的干涉公式为： $I(P) = I_0(1+ gamma cos delta (P))$，其中 $I_0 = I_1 + I_2$
