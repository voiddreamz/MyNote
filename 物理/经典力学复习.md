## 转动
1. 我们在转动中和平动中存在这样的一个对应关系式

|      | 平动                                | 转动                             | 转换                            |
| ---- | --------------------------------- | ------------------------------ | ----------------------------- |
| 位置   | $x$                               | $theta$                        | ${} x=r theta {}$             |
| 速度   | ${} v=(d x)/(d t) {}$             | ${} omega =(d theta)/(d t) {}$ | ${} v=r omega  {}$            |
| 加速度  | ${} a=(d v)/(d t) {}$             | ${} alpha =(d omega)/(d t) {}$ | ${} a = r alpha {}$           |
| 广义质量 | 物体相对惯性坐标系运动，运动方程中的一个常量，描述动能变化的大小。 | 转动惯量 $I$                       |                               |
| 动力   | 在空间中 Lagrange 量变化的体现，破坏了势能空间的均匀性。 | 力矩 $tau=I alpha$               | $tau=arrow(r) times arrow(F)$ |
| 动量   | ${} p=m v {}$                     | 角动量 ${} L {}$                  | $L = I omega$                 |
