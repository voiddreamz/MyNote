## 第1章 MATLAB基础

### 向量与矩阵
```matlab
a = [3, 5, 7, 8]             % 行向量（逗号或空格分隔）
b = a'                        % 转置为列向量
c = 1:2:13                    % 1到13，步长2 (start:step:end)
d = linspace(0, 2*pi, 20)     % 0到2π间等距20个点
e = logspace(1, 3, 50)        % 10^1到10^3间等对数间隔50个点
A = [2, 4, 5; 9, 7, 1]       % 矩阵，分号换行
```

### 逻辑运算
```matlab
A = [1, 3, -1, 0, 7, 2]
B = ~(A>2)                    % 不大于2的元素位置 → 1 0 1 1 0 1
C = (A>0) & (A<3)             % 大于0小于3 → 1 0 0 0 0 1
D = A.*B                      % 逐元素乘法（数组运算）
```

### 三种定义函数的方式
```matlab
% 方式1：匿名函数 @ （最常用）
f = @(x, t) cos(x.^2).^2 + t
f(3, 2.1)                     % ans = 2.9302

% 方式2：inline（旧版）
f = inline('cos(x.^2).^2 + t', 'x', 't')
f(3, 2.1)

% 方式3：函数文件（独立.m文件）
function F = test2(x, theta)
    F = cos(x.^2).^2 + theta;
end
```

### 三维作图
```matlab

plot3(x, y, z, '-.');  view(-60, 18)

```

### 动画（句柄控制）

## 第2章 迭代—分形

### Sierpinski三角形（递归）
```matlab
function sjx(A1, A2, A3, N1)       % 递归函数标准格式
    B1 = (A1+A2)/2; B2 = (A1+A3)/2; B3 = (A2+A3)/2;
    N2 = N1 - 1;
    if N2 > 0
        sjx(A1, B1, B2, N2);       % 递归调用
        sjx(B1, A2, B3, N2);
        sjx(B2, B3, A3, N2);
        fill(real([A1,A2,A3,A1]), imag([A1,A2,A3,A1]), 'g');
        hold on
        fill(real([B1,B2,B3,B1]), imag([B1,B2,B3,B1]), 'w');
    end
end
```

### Julia集（复变迭代）

### Koch雪花
```matlab
new = [0, i, sqrt(3)/2+i/2, 0];  % 初始三角形
for k = 1:8
    old = new;
    diff = (old(2:end) - old(1:end-1)) / 3;
    new(1:4:4*length(old)-7) = old(1:end-1);
    new(2:4:end) = old(1:end-1) + diff;
    new(3:4:end) = new(2:4:end) + diff*(0.5+0.5*sqrt(3)*i);
    new(4:4:end) = old(1:end-1) + 2*diff;
    new(end+1) = old(end);
    plot(new);  axis equal
end
```

## 第3章 数值微分与积分

### diff — 差分
```matlab
a = [1, 4, 5, -2];
diff(a)              % ans = 3  1  -7  （后项减前项）
diff(b)              % 对矩阵：默认按列作差分
```

### gradient — 梯度（中心差分）
```matlab
a = [1, 5, 4, 2, 7; 2, 1, 8, 3, 4]
[px, py] = gradient(a)   % px=行方向偏导, py=列方向偏导
```

### del2 — 离散拉普拉斯
```matlab
[x,y] = meshgrid(-2:2, -3:3);
U = x.*x + y.*y;
V = 4 * del2(U)          % 对U=x^2+y^2，结果全为4
```

### trapz — 梯形积分
```matlab
a = [1, 5, 7, 2, 3];  trapz(a)   % ans = 16
trapz(b)                          % 矩阵默认按列积分
trapz(b, 2)                       % 指定按行积分
```

### quad / quadl — 函数积分
```matlab
% quad: 自适应Simpson  |  quadl: Lobatto法（精度更高）

f = @(x) exp(x.^(-3));
quad(f, 0.5, 0.6)            % ans ≈ 71.5465

% 含参数积分（参数跟在区间后）
quad(@betaf, 0, 1, 1e-6, z, w)  % format: quad(fun, a, b, tol, p1, p2...)
```

### 多重积分
```matlab
dblquad(fun, xmin, xmax, ymin, ymax)         % 二重
triplequad(@(x,y,z) y*sin(x)+z*cos(x), 0, pi, 0, 1, -1, 1)  % 三重
```

## 第4章 数据处理

### interp1 — 一维插值
```matlab
yi = interp1(x, y, xi, 'method')
% method: 'nearest' | 'linear' | 'spline' | 'pchip'

x = 0:10;  y = sin(x);
xi = 0:.15:10;
yi = interp1(x, y, xi, 'spline');
plot(x, y, 'o', xi, yi, '-')
```

### polyfit / polyval — 多项式拟合
```matlab
P = polyfit(x, y, n)       % n次拟合，P为系数（降幂排列）
y_fit = polyval(P, x)      % 计算拟合值

t = 0:0.5:5;  s = 1*t + 0.2.*t.*t;   % 真实二次曲线
ss = s + 0.3*randn(size(t));          % 加噪声
P = polyfit(t, ss, 2);
y = polyval(P, t);
plot(t, ss, 'r*', t, y, 'b-')
```

### fzero — 求单变量零点
```matlab
x = fzero(@sin, 3)                           % 函数句柄
x = fzero(@(x) x.^2-2, 1)                    % 匿名函数
```

### roots — 多项式全部根
```matlab
roots([5, 3, -4, 1, 7, 9])    % 系数降幂
```

### fminbnd — 单变量求极小值
```matlab
f = @(x) -1./((x-.3).^2+.01) - 1./((x-.9).^2+.04) + 6;
x_min = fminbnd(f, 0, 0.4)    % 区间内找极小值
```

### fft — 快速傅里叶变换
```matlab
Y = fft(x)          % 一维FFT
y = ifft(Y)         % 逆变换

% 功率谱密度
Y = fft(y, 256);
Pyy = Y .* conj(Y) / 256;
f = 1000/256 * (0:127);
plot(f, Pyy(1:128))
```

## 第5章 解常微分方程

### ode45 — 把二阶ODE化为一阶组
```matlab
% y'' = -y   →   设 y1=y, y2=y'
% 方程组: y1' = y2,  y2' = -y1

% 写法1：匿名函数
F = @(t, y) [y(2); -y(1)]
[T, Y] = ode45(F, [0, 10], [0, 2])    % [0,2] = 初值[y1(0),y2(0)]

% 写法2：子函数文件
function main
    [T, Y] = ode45(@myode, [0, 10], [0, 2]);
    plot(T, Y(:,1))
end
function ydot = myode(t, y)
    ydot = [y(2); -y(1)];
end
```

### ode求解器选择
| 指令 | 适用 | 方法 |
|------|------|------|
| `ode45` | **非刚性**（首选） | 四五阶RK |
| `ode23` | 非刚性 | 二三阶RK |
| `ode15s` / `ode23s` | **刚性**问题 | 刚性专用 |

### odeset — 选项设置
```matlab
options = odeset('RelTol', 1e-6)                     % 控制精度
options = odeset('OutputFcn', 'odephas2')             % 输出相图
sol = ode45(F, [0:0.1:10], [0, 1])                    % 输出结构体
ss = deval(sol, 5.15)                                  % 在t=5.15处插值
```

### 事件 (Events)
```matlab
function [value, isterminal, direction] = events(T, Y)
    value = Y(1);            % value=0时事件触发
    isterminal = 1;           % 1=停止积分
    direction = 0;            % 0=双向检测
end
opts = odeset('events', @events);
[t, y] = ode45(@f, [0, Inf], y0, opts);
```

### bvp4c — 边值问题
```matlab
solinit = bvpinit(x, v)                     % 初始猜测解
sol = bvp4c(@odefun, @bcfun, solinit)       % 求解
```

## 速记表

| 概念 | 要点 |
|------|------|
| 数组 vs 矩阵运算 | `.* ./ .^` 逐元素；`* / ^` 线性代数 |
| 匿名函数 @ | `f = @(x) x.^2` — 首选 |
| 数据网格 | `[X,Y] = meshgrid(x,y)` — 三维绘图第一步 |
| 句柄动画 | set + EraseMode + drawnow |
| 递归 | 函数调用自身，需终止条件 |
| 积分 | trapz(离散数据) / quad(连续函数) |
| 拟合 vs 插值 | polyfit可以不过数据点 / interp1必须过 |
| 刚性ODE | 解变化剧烈 → ode15s/ode23s 替代 ode45 |
| 化高阶ODE | y1=y, y2=y', ... → 一阶方程组 |
