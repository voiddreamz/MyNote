### 二、 导数

1. 对于求复杂难以计算的函数的导数，尝试先使用定义法。

1. 遇到上下有分式，分母为x后无思路时，可尝试看作 $frac(f(x) - f(0), x - 0)$，后便可用倒数思维以及拉格朗日中值定理去解题

    如此题：$x e^(x_(n+1)) = e^(x_n) - 1$

    计算过程中会出现 $e^(x_(n+1)) = frac(e^(x_n) - 1, x)$

    便可视为 $e^(x_(n+1)) = frac(e^(x_n) - e^0, x - 0)$

    后使用拉格朗日定理看作 $e^(x_(n+1)) = frac(e^(x_n) - 1, x) < e^xi < e^(x_n)$

    则其单调性得到证明，再证明其大于0即可

2. 对于判断存在绝对值是否可导的问题，尝试将其先化作分段函数。
3. 关于含有绝对值的函数是否可导，存在以下两个结论：
    (1) $f(x) = abs(x - a)$ 在 $x = a$ 处不可导
    (2) 若 $f(x) = varphi(x) abs(x - a)$ 在 $x = a$ 处可导，则 $varphi(a) = 0$

4. 对于下题，设 $f(0) = 0$，则 $f(x)$ 在 $x = 0$ 处可导的一个充分条件是：
    (A) $lim_(x -> 0) frac(f(3 h) - f(h), 2 h)$ 存在
    (B) $lim_(x -> 0) frac(f(h) - f(-h), 2 h)$ 存在
    (C) $lim_(x -> 0) frac(f(h^3), h^2)$ 存在
    (D) $lim_(x -> 0) frac(f(h^2), h^2)$ 存在

    对于 ABD 选项，设 $f(x) = x^2 + 1 (x != 0), f(x) = 0 (x = 0)$，则显然其于 $x = 0$ 处不可导，其对于 AB 选项存在导数定义的极限。对于 D 选项只能证明存在右极限。

5. 对洛必达法则适应的形式，务必别忘了先使用等价无穷小，后使用洛必达。且一定要对于 $f(x) - 0$ 敏感，其或许为 $frac(f(x) - f(0), x - 0)$（或者自行乘上其倒数凑成导数的定义）

6. 对于遇到三次方以上仅知斜率求未知切线的问题，可先根据斜率相同求出切点后求出其切线。
7. 对于下题：已知 $f(x - frac(1, x)) = frac(x^4 + 1, x^4 + x^2 + 1)$，求 $f'(x)$。可将其凑为 $frac(x^2 + frac(1, x^2), x^2 + 1 + frac(1, x^2))$，便可凑出 $x - frac(1, x)$，进行 $f'(x)$ 的计算

8. 对于诸如 $f(x) = (x^2 + 1)^5 (x^5 + 1)^2$，求 $f^((20))(x)$。观察其最高次项为 $x^2$，即可直接给出答案为 $20!$。由此可以得出结论，存在高阶导数时，直接观察 x 的最高次项，若高于此则为 0，低于此则找规律计算。

9. 对于含有 $sin^4 x + cos^4 x$ 的式子，将其化作 $(sin^2 x + cos^2 x)^2 - "某"$

    如求 $f(x) = sin^4 x + cos^4 x$ 的 n 阶导数：
    解：$f(x) = (sin^2 x + cos^2 x)^2 - 2 (sin x cos x)^2 = 1 - frac(sin^2 2 x, 2) = frac(3, 4) + frac(cos 4 x, 4)$
    则其 n 阶导数为 $f^((n))(x) = 4^(n-1) cos(4 x + frac(n pi, 2))$

10. 诸如对于求 $lim_(x -> x_0) frac(1, 1 + (1 + frac(2, 3 n))^(3 n))$ 的题，其可化为 $frac(1, 1 + lim_(x -> x_0) (1 + frac(2, 3 n))^(3 n))$

11. $lim_(x -> infinity) root(n)(a^n + b^n + c^n + dots) = max{a, b, c, dots}$

12. 对于相似于导数定义的式子，但是有些不同的式子，如：$lim_(h -> 0) frac(f(x_0 + 3 h) - 3 f(x_0 - h), h) = 1$，求 $f'(x_0)$。则可将其拆开后凑成定义式。由分母趋近于 0，则分子趋近于 0，则 $f(x_0) = 0$。原式 $= lim_(h -> 0) frac(f(x_0 + 3 h), h)$

13. 对于含有导函数的零点也即是中值定理的应用，存在以下的一些题型：
    (1) 仅含有两项，导函数的系数包含自变量，此时可将其除至另一端，变为 $f'(x) - frac(a, c x + b)$ 从而求其不定积分
    (2) 对于求 $f''(x)$ 的导数，但是无法得出 $f'(x) = 0$ 的另一个端点，可尝试看看在原函数中是否存在 $xi_1$ 使得一阶导数满足 $f'(xi_1) = 0$，则其一般一个端点为题目中两端点其一，另一端点为 $xi_1$
    (3) 对于部分非纯正的 $f'(x)$，可尝试使用构造函数。
    (4) 对于题目中信息较少的题，尝试自行设出 $x_0$，使得 $f(x_0) = f(a)$（题目中已知）。如下题：$y = f(x)$ 在 $[0, 3]$ 上连续，$(0, 3)$ 上可导，$f(0) + f(1) + f(2) = 3, f(3) = 1$，求证存在一点 $xi in (0, 1)$，使得 $f'(xi) = 0$。由闭区间的连续函数性质可得 $m <= f(0), f(1), f(2) <= M$，则 $3 m <= f(0) + (1) + f(2) = 3 <= 3 M$，则由介值定理可得出存在 $c in (0, 2)$，使得 $m < f(c) = 1 < M$，由罗尔定理, 后略.

14. 关于导数（微商）于微分，涉及到反函数二阶导时，如下题：设 $x = varphi(y)$ 为 $y = x + ln x$ 的反函数，求二阶导数 $frac(dif^2 x, dif y^2)$

    存在以下两种解法：
    (1) 由反函数求导法则，易得 $frac(dif x, dif y) = frac(x, x + 1)$
    则 $frac(dif^2 x, dif y^2) = frac(dif dot dif x, dif y dot y) frac(dif dot dif x, dif y dot dif y) = frac(dif (frac(dif x, dif y)), dif y) = frac(dif (frac(x, x + 1)), dif x) dot frac(dif x, dif y) = frac(1, (x + 1)^2) dot frac(x, x + 1) = frac(x, (x + 1)^3)$
    (2) 使用隐函数的求导法则(必会的方法，重要)
    两边对 y 求导，课得到：$1 = frac(dif x, dif y) + frac(1, x) dot frac(dif x, dif y)$
    两侧再次求导可得到：$0 = frac(dif^2 x, dif y^2) - frac(1, x^2) dot (frac(dif x, dif y))^2 + frac(1, x) dot frac(dif^2 x, dif y^2)$
    则可求得所求。

5. 对于如 $x^3, x^1, x^(-1)$ 等奇数同时存在于分母分子的，可将其同时除以一偶数次方

    如此题：$integral frac(e^(3 x) + e^x, e^(2 x) + e^(4 x) + 1) dif x$
    可上下同时除以 $e^(2 x)$，使其成为 $integral frac(e^x + e^(-x), e^(-2 x) + e^(2 x) + 1) dif x$
    分子可化为 $dif (e^x - e^(-x))$，分母化为 $(e^x - e^(-x))^2 + 1$

6. 对于分母存在 $1 plus.minus sin x / cos x$，可上下同时乘 $1 plus.minus sin x / cos x$

7. 对于分子是 x 的一次项，分母是 x 的二次项，可将分母求导，将分子转化后化入 dif x

    如此题：$integral frac(12 x + 9, 2 x^2 - 3 x - 9) dif x$
    分子可化为 $3 (4 x - 3) + 6$
    则原式化为 $3 integral frac(dif (2 x^2 - 3 x - 9), 2 x^2 - 3 - 9) + 6 integral frac(dif x, 2^X - x x x x x x x)$，便为 arctan x 或的形式
