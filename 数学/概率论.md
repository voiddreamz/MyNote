### 一、事件间的关系及运算：
$A union B = A + B, A inter B = A B, A - B = A overline(B)$
交换律结合律和数相同，其分配律与数的四则运算不同，满足以下规则：
1. $A union (B inter C) = A + B C = (A + B) (A + C) = (A union B) inter (A union C)$
2. $A inter (B union C) = A (B + C) = A B + A C = (A inter B) union (A inter C)$
1式主要用于方便计算，例如：
	例1. $(A union B) (A union overline(B)) (overline(A) union B)$
	我们将其换算为 $(A + B) (A + overline(B)) (overline(A) + B)$。由于前两项都含有A，则利用式1将其转换，后续同理则可得到答案为 $A B$
**对偶律（摩尔根律）**：$overline(A union B) = overline(A) inter overline(B), overline(A inter B) = overline(A) union overline(B)$
其推广到整个事件：$overline(union_(i=1)^(infinity) A_i) = inter_(i=1)^(infinity) overline(A_i)$

### 二、古典概型
**概念**：(1) 试验E的样本空间 $Omega$ 只包含有限个样本点。(2) 每个样本点出现的可能性相同
**公式**：$P(A) = frac(k, n)$
#### 排列组合复习：
1. 排列：
	从n个不同元素中取出m个元素，按照一定顺序排成一列
	计算公式：$A_n^m = frac(n!, (n - m)!)$
	例：$A_7^3 = 7 dot 6 dot 5$
2. 组合：
	从n个不同元素中，任取m个不同元素组成一组
	计算公式：$C_n^m = frac(n!, m! (n - m)!) = frac(n (n - 1) dots (n - m + 1), m!)$
3. 性质：
	$C_n^m = C_n^(n - m); C_n^1 = n; C_(n+1)^m = C_n^m + C_n^(m - 1)$
