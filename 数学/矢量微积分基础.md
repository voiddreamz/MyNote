### 小 tips
1. 遇到积分完全不知道它在算什么的时候，可以尝试将它分解为单位矢量乘以长度，例如对于 $integral.cont_C (A dot dif l)$，A 是矢量场，d l 可以分解后其单位矢量和 A 相乘即为 A 在 d l 方向上的分量乘以 d l 的长度。通过积分将A 在 d l 方向上的分量乘以 d l 的长度转了一圈，如果是力场，则为力对粒子绕一圈所做的功；如为流速场，则为环流速。
2. 无旋 = 保守场，无散=无源场
3. $m frac(dif^2 x, dif t^2) + k x = 0$
4. $mat(dif y_1 / dif t, dif y_2 / dif t) = mat(y_2, - (k/m) y_1)$
5. $mat(dif y_1 / dif t, dif y_2 / dif t) = mat(y_2, - (k/m) y_1)$
6. $$epsilon_{i j k} = 
+1 & #text("如果") (i,j,k) text{ 是 } (1,2,3) #text{ "的偶排列"} \\
-1 & #text{如果 } (i,j,k) \text{ 是 } (1,2,3) #text{ 的奇排列} \\
0 & #text{如果有任意两个指标相同}
\end{cases}$$
其中偶排列为按照 ijk 顺序轮换例如 312，奇排列为交换一次顺序例如 132
