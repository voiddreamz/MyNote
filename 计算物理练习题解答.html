第1题 斜坡上的斜抛（空气阻力与v成正比）

物理：物体在含阻力的重力场中运动，落在斜坡上。数学：含阻力ODE+事件检测着陆。
% ====== 第1题：斜坡上的斜抛运动（空气阻力 f = -kv）======

clear;                          % 清空工作区所有变量，防止旧数据干扰
close all;                      % 关闭所有图形窗口
g = 9.8;                        % 重力加速度 (m/s²)
m = 1;                          % 物体质量 (kg)
k = 0.1;                        % 空气阻力系数 (kg/s)，阻力 = -k*v
v0 = 20;                        % 初速度大小 (m/s)

alpha = 30 * pi/180;            % 山坡仰角，30度转弧度（pi/180是把度转弧度的系数）

% 生成待测试的射角范围：从比坡度大5°开始，到80°，每隔1°取一个
theta_list = (alpha + 5*pi/180) : (1*pi/180) : (80*pi/180);

R = zeros(size(theta_list));    % 预分配一个和theta_list同样大小的数组，用来存每个射角对应的射程

for j = 1:length(theta_list)       % 对每一个射角做计算
    theta = theta_list(j);          % 当前射角（弧度）
    
    % 定义ODE方程组。MATLAB解ODE需化为一阶组：y(1)=x, y(2)=vx, y(3)=y, y(4)=vy
    odefun = @(t,y) [...
        y(2);                       % dx/dt = vx（水平速度就是位置的变化率）
        -k/m * y(2);                % dvx/dt = -(k/m)*vx（阻力使水平速度减小）
        y(4);                       % dy/dt = vy
        -g - k/m * y(4)             % dvy/dt = -g - (k/m)*vy（重力+阻力使竖直速度减小）
    ];
    
    % odeset设置"事件"：当y(3)-y(1)*tan(alpha)=0时停止，即物体落到斜坡上
    opts = odeset('Events', @(t,y) deal(...
        y(3)-y(1)*tan(alpha),       % 条件：y坐标 - x坐标*tan(坡度) = 0
        1,                           % isterminal=1: 满足条件就停止积分
        -1));                        % direction=-1: 只在上往下穿越时触发
    
    % ode45求解：时间0~30s（足够长了），初值[x0,vx0,y0,vy0]
    [~, Y] = ode45(odefun, [0, 30], [0, v0*cos(theta), 0, v0*sin(theta)], opts);
    
    % Y的最后一行就是落地时的状态，Y(end,1)是落地时x坐标
    % 沿斜坡的射程 = x / cos(坡度)  （因为x=R*cosα）
    R(j) = Y(end,1) / cos(alpha);
end

% 找到最大射程对应的位置
[R_max, idx_max] = max(R);                              % max返回最大值和它的位置索引
fprintf('α=30°时，最佳射角 = %.1f°，最大射程 = %.2f m\n', ...
    theta_list(idx_max)*180/pi, R_max);                 % 把弧度转回度来显示

% 画射程 vs 射角的曲线
figure('Name','第1题');                              % 创建新图形窗口并命名
plot(theta_list*180/pi, R, 'b-', 'LineWidth', 2);   % 画蓝色实线，线宽2
xlabel('射角 θ (°)');                              % x轴标签
ylabel('射程 R (m)');                               % y轴标签
title('斜坡α=30°时射程与射角关系');                  % 标题
grid on;                                                % 打开网格线
关键模板：含阻力抛体 → 用 ode45 + Events 检测着陆
初值写法：[x0, vx0, y0, vy0] = [0, v0·cosθ, 0, v0·sinθ]
事件函数：返回 [条件, 是否停止(1=是), 穿越方向(0=双向,-1=向下)]

第2题 斜抛的射程与射高（椭圆性质验证）

物理：无阻力时斜抛最高点分布在椭圆上。含阻力时数值求解ODE。
% ====== 第2题：水平射程、竖直射高、椭圆验证 ======
clear; close all;
g = 9.8; v0 = 20; m = 1; k = 0.05;           % 参数定义

theta = (10:5:80) * pi/180;                   % 10°到80°，每5°一个射角
x_range = zeros(size(theta));                 % 预分配：存水平射程
y_max   = zeros(size(theta));                 % 预分配：存竖直射高

figure('Name','第2题');
for i = 1:length(theta)
    th = theta(i);
    % ODE：和第一题完全一样
    odefun = @(t,y) [y(2); -k/m*y(2); y(4); -g - k/m*y(4)];
    
    % 这次事件检测 y=0（落地）
    opts = odeset('Events', @(t,y) deal(y(3), 1, -1));
    
    % 求解
    [~, Y] = ode45(odefun, [0, 30], [0, v0*cos(th), 0, v0*sin(th)], opts);
    
    x_range(i) = Y(end,1);                      % 最后一行第1列=落地x坐标=射程
    y_max(i) = max(Y(:,3));                     % 第3列(y坐标)的最大值=射高
    
    plot(Y(:,1), Y(:,3)); hold on;              % 画整条轨迹，hold on使多条线叠加
end
xlabel('x (m)'); ylabel('y (m)'); title('含阻力斜抛轨迹'); grid on;

% ==== 以下单独验证无阻力时的椭圆性质 ====
figure('Name','椭圆验证');
theta2 = (1:89) * pi/180;                       % 1°~89°，几乎覆盖所有射角
x_h = v0^2 * sin(2*theta2) / (2*g);             % 无阻力：x_H = v0²·sin2θ/(2g)
y_h = v0^2 * sin(theta2).^2 / (2*g);            % 无阻力：y_H = v0²·sin²θ/(2g)
plot(x_h, y_h, 'b.'); hold on;                   % 画所有最高点（蓝点）

a = v0^2/(4*g);                                 % 椭圆的半长轴
th_ell = linspace(0, 2*pi, 200);                % 0到2π，200个点画椭圆
plot(a*cos(th_ell), a + a*sin(th_ell), 'r--');   % 画椭圆虚线
xlabel('x'); ylabel('y'); axis equal; grid on;        % axis equal使x和y轴比例一致
title('最高点分布在椭圆上(无阻力)');
关键模板：无阻力抛体 → 解析公式直接算；有阻力 → ode45
最高点椭圆：(x_H/a)²+((y_H-a)/a)²=1, a=v₀²/(4g)

第3题 飞机在风中往返时间

物理：速度合成——空速+风速=地速。往返时间受风向影响。
% ====== 第3题：飞机风中往返 ======
clear; close all;
l = 1000;                                       % AB距离 (km)
v = 500;                                        % 飞机相对空气的速度 (km/h)
u = 100;                                        % 风速 (km/h)

theta = linspace(0, 2*pi, 720);                 % 风向角度，0到360度，720个点

sin_phi = u * sin(theta) / v;                   % 航向修正角φ满足 sinφ = u·sinθ/v
cos_phi = sqrt(1 - sin_phi.^2);                 % 由sin求cos

% 去程地速：顺风分量 v*cosφ + u*cosθ
t_go = l ./ (v * cos_phi + u * cos(theta));     % ./ 是逐元素除法。去程时间=距离/去程地速
% 回程地速：逆风分量 v*cosφ - u*cosθ
t_back = l ./ (v * cos_phi - u * cos(theta));   % 回程时间
t_total = t_go + t_back;                        % 往返总时间

figure('Name','第3题');
polarplot(theta, t_total);                      % 极坐标图：角度=风向，半径=时间
title(sprintf('往返时间 vs 风向'));

[t_min, i_min] = min(t_total);                  % 找最小时间和对应位置
[t_max, i_max] = max(t_total);                  % 找最大时间和对应位置
fprintf('最短 %.2fh (风向 %.0f°), 最长 %.2fh (风向 %.0f°)\n', ...
    t_min, theta(i_min)*180/pi, t_max, theta(i_max)*180/pi);
关键模板：速度合成问题 → 直接解析计算，不用ODE
技巧：min/max找极值，polarplot画极坐标

第4题 链条从桌面滑落

物理：变质量系统的牛顿第二定律。下垂部分的重量驱动整条链加速。
% ====== 第4题：链条从光滑桌面滑落 ======
clear; close all;
l = 1.0;  b = 0.2;  g = 9.8;                   % 链长、初始下垂长度、重力加速度

% 设下垂长度x(t)，整条链质量m
% 下垂部分质量 = (x/l)·m，受重力 = (x/l)·m·g
% 牛顿第二定律：m·x'' = (x/l)·m·g → x'' = (g/l)·x
% 这个方程的解是双曲余弦函数：x(t) = b·cosh(√(g/l)·t)

omega = sqrt(g/l);                              % 特征频率 ω = √(g/l)

% 解析解——滑出桌面（x=l）的时刻
t_slide = acosh(l/b) / omega;                   % acosh是反双曲余弦，cosh的反函数
v_slide = b*omega * sinh(omega*t_slide);        % sinh是双曲正弦

fprintf('滑出时间=%.4fs, 速度=%.4fm/s\n', t_slide, v_slide);

% 用ode45数值验证：
odefun = @(t,y) [y(2); g/l*y(1)];               % y(1)=x,y(2)=v。x''=(g/l)*x
opts = odeset('Events', @(t,y) deal(y(1)-l,1,1)); % x=l时停止
[t_num, Y] = ode45(odefun, [0, t_slide*1.2], [b, 0], opts);

t_plot = linspace(0, t_slide, 200);             % 0到滑出时刻，200个等间隔点
x_ana = b * cosh(omega*t_plot);                 % 解析解位置
v_ana = b*omega * sinh(omega*t_plot);           % 解析解速度

figure('Name','第4题');
plot(t_plot, x_ana, t_num, Y(:,1), 'ro');         % 解析解(线)+数值解(红圈)对比
xlabel('t(s)'); ylabel('x(m)'); legend('解析','数值'); grid on;
关键模板：变质量力学 → 先写牛顿方程再化为一阶ODE
常见形式：x'' = (常数)·x → 解为双曲函数 cosh/sinh

第5题 物体从半球面滑下（脱离角度）

物理：动量守恒+能量守恒→导出脱离条件→fzero求脱离角度。
% ====== 第5题：半球面滑落的脱离角度 ======
clear; close all;
R = 1;  g = 9.8;

% 列出要测试的几种质量比 m/M
mr_list = [0.1, 0.5, 1, 2, 5, 10];             % m是物体B质量，M是半球A质量
theta_detach = zeros(size(mr_list));            % 预分配存脱离角

for idx = 1:length(mr_list)
    mr = mr_list(idx);                          % 当前质量比
    
    % 由水平动量守恒和能量守恒导出脱离时的条件方程：
    % 2*(1-cosθ)*(1+mr) - cosθ*(1+mr*sin²θ) = 0
    % 把这个方程定义为匿名函数，准备让fzero来求根
    func = @(th) 2*(1-cos(th))*(1+mr) - cos(th)*(1+mr*sin(th)^2);
    
    % fzero在区间[0.1, π/2-0.1]内搜索使func=0的θ
    % 区间两个端点函数值符号必须相反才能保证有根
    theta_detach(idx) = fzero(func, [0.1, pi/2-0.1]);
end

figure('Name','第5题');
semilogx(mr_list, theta_detach*180/pi, 'b-o');  % semilogx:x轴对数坐标,y轴线性
xlabel('m/M'); ylabel('脱离角(°)'); grid on;

fprintf('m/M→0(固定半球): 脱离角=%.2f°\n', acos(2/3)*180/pi);
关键模板：约束运动脱离 → fzero求脱离条件方程的根
fzero用法：fzero(@方程, [下限, 上限]) — 端点函数值必须异号

第6题 受迫振动（共振曲线）

物理：简谐振子在周期性外力下的稳态响应。核心：共振频率、振幅-频率曲线。
% ====== 第6题：阻尼受迫振动——共振 ======
clear; close all;
m = 1;  k = 10;                                  % 质量，劲度系数
omega0 = sqrt(k/m);                              % 固有角频率 ω₀=√(k/m)
F0 = 1;                                          % 驱动力幅值

gamma_vals = [0.5, 1, 2, 4];                    % 四种阻尼系数，阻尼越大共振越不明显
Omega = linspace(0.1, 3*omega0, 500);            % 驱动频率从0.1ω₀扫到3ω₀

figure('Name','第6题');
for i = 1:length(gamma_vals)
    g = gamma_vals(i);
    beta = g/(2*m);                              % 衰减系数 β=γ/(2m)
    
    % 稳态位移振幅公式：A = F₀/√[(k-mΩ²)² + (γΩ)²]
    A = F0 ./ sqrt((k - m*Omega.^2).^2 + (g*Omega).^2);
    
    % 速度振幅 = Ω*A。速度共振始终在Ω=ω₀处
    V_amp = Omega .* A;
    
    subplot(2,2,i);                              % 2行2列的第i个图
    yyaxis left;                                 % 使用左侧y轴（蓝色）
    plot(Omega/omega0, A, 'b-');                  % 位移振幅曲线
    ylabel('位移振幅 A');
    yyaxis right;                                % 使用右侧y轴（红色）——同一张图两个不同y轴
    plot(Omega/omega0, V_amp, 'r--');             % 速度振幅曲线
    ylabel('速度振幅');
    xlabel('\Omega/\omega_0');                   % x轴是驱动频率/固有频率的比值
    title(sprintf('\\gamma=%.1f', g));
    grid on;
end
yyaxis left/right：同一子图用左右两个不同的y轴，分别画位移振幅（左轴）和速度振幅（右轴）
共振规律：速度共振恒在Ω=ω₀，位移共振Ωr<ω₀且随阻尼增大左移

第7题 阻尼振动（三种阻尼对比）

物理：无外力的衰减振动。特征方程判别阻尼类型：欠阻尼(振荡衰减)、临界阻尼(最快回平衡)、过阻尼(慢衰减)。
% ====== 第7题：三种阻尼振动对比 ======
clear; close all;
m = 1;  k = 10;  omega0 = sqrt(k/m);             % 固有频率
A0 = 1;                                          % 初始位移 1m，从静止释放(初速=0)
t = linspace(0, 20, 2000);                       % 时间 0~20s，2000个点保证曲线光滑

% 四种情况：β(衰减系数)分别为0.2ω₀、0.5ω₀、1.0ω₀、1.5ω₀
beta_vals = [0.2, 0.5, 1.0, 1.5] * omega0;
labels = {'欠阻尼(轻)','欠阻尼(重)','临界阻尼','过阻尼'};

figure('Name','第7题');
for i = 1:4
    beta = beta_vals(i);
    
    if beta < omega0                            % ||| 欠阻尼：β<ω₀ |||
        omega_d = sqrt(omega0^2 - beta^2);      % 阻尼振荡频率（比ω₀小）
        x = A0 * exp(-beta*t) .* (cos(omega_d*t) + beta/omega_d * sin(omega_d*t));
        % 振幅按e^{-βt}指数衰减，振荡频率ω_d
        
    elseif abs(beta - omega0) < 1e-6            % ||| 临界阻尼：β=ω₀ |||
        x = A0 * (1 + omega0*t) .* exp(-omega0*t);
        % 刚好不振荡，以最快速度回到平衡位置
        
    else                                        % ||| 过阻尼：β>ω₀ |||
        lam1 = -beta + sqrt(beta^2 - omega0^2); % 特征根1（绝对值较小）
        lam2 = -beta - sqrt(beta^2 - omega0^2); % 特征根2（绝对值较大）
        C1 = A0 * lam2/(lam2 - lam1);           % 由初位移A0初速0确定系数C1
        C2 = A0 * lam1/(lam1 - lam2);           % 系数C2
        x = C1*exp(lam1*t) + C2*exp(lam2*t);    % 两个指数衰减的叠加
    end
    
    subplot(2,2,i);
    plot(t, x);  xlabel('t(s)'); ylabel('x(m)');
    title(labels{i}); grid on;
end
判别条件：β<ω₀→欠阻尼振荡；β=ω₀→临界阻尼；β>ω₀→过阻尼
记忆：β=γ/(2m)衰减系数，ω₀=√(k/m)固有频率

第8题 双缝干涉

光学：两束相干光叠加，强度 I=2I₀(1+cosΔφ)。条纹间距 Δy=λL/d。
% ====== 第8题：双缝干涉 ======
clear; close all;
lambda = 500e-9;                                % 波长 500nm=5e-7m
k = 2*pi / lambda;                              % 波数 k=2π/λ
I0 = 1;                                         % 单缝光强（归一化为1）
d = 0.001;                                      % 双缝间距 1mm=0.001m
L = 1;                                          % 缝到屏的距离 1m
y = linspace(-0.005, 0.005, 2000);              % 屏上坐标 ±5mm，2000个点

delta_r = d * y / L;                            % 光程差（远场近似）：Δr≈d·sinθ≈d·y/L
delta_phi = k * delta_r;                        % 相位差 Δφ = (2π/λ)·Δr

% 干涉光强公式：I = I₁+I₂+2√(I₁I₂)cosΔφ = 2I₀(1+cosΔφ)
I = 2 * I0 * (1 + cos(delta_phi));

figure('Name','第8题');
subplot(2,1,1);                                 % 2行1列，画在第1个位置
plot(y*1000, I);                                % x用mm单位(×1000)
xlabel('y(mm)'); ylabel('光强'); title('双缝干涉光强分布'); grid on;

subplot(2,1,2);                                 % 画在第2个位置
imagesc(y*1000, [0,1], repmat(I/max(I),[20,1]));% repmat:把一行光强重复20行→二维条纹图
colormap(gray);                                 % 用灰度colormap，白=亮，黑=暗
xlabel('y(mm)'); title('干涉条纹图样');

fprintf('条纹间距 Δy = λL/d = %.3f mm\n', lambda*L/d*1000);
关键模板：干涉/衍射 → 光强公式直接算，不需要ODE
imagesc+repmat：把一维数据复制成二维来显示条纹图样
colormap(gray)：设定颜色映射为灰度

第9题 单缝衍射

光学：I=I₀[sinβ/β]², β=πa·sinθ/λ。半波带法判断明暗纹。
% ====== 第9题：夫琅禾费单缝衍射 ======
clear; close all;
lambda = 500e-9;  L = 1;                        % 波长、距离
y = linspace(-0.1, 0.1, 2000);                  % 屏上坐标 ±0.1m
theta = atan(y/L);                              % atan是反正切，由y和L算出衍射角θ

a_vals = [2, 5, 10] * 1e-6;                     % 三种缝宽：2μm, 5μm, 10μm

figure('Name','第9题');
for i = 1:3
    a = a_vals(i);
    beta = pi * a * sin(theta) / lambda;        % 单缝衍射的特征参数 β
    I = (sin(beta+1e-12)./(beta+1e-12)).^2;     % sinc²(β)，+1e-12防止除以0
    
    subplot(2,2,i);
    plot(y*1000, I);                            % 横轴单位mm
    xlabel('y(mm)'); ylabel('I/I_0');
    title(sprintf('a=%d\\mum', round(a*1e6)));
    grid on;
end

% 第4个子图：同一缝宽下三种波长对比
subplot(2,2,4);
a = 5e-6;
for lam = [750, 540, 440]*1e-9               % 依次红、绿、蓝光
    beta = pi * a * sin(theta) / lam;
    I = (sin(beta+1e-12)./(beta+1e-12)).^2;
    plot(y*1000, I); hold on;                   % 三条曲线叠加
end
xlabel('y(mm)'); legend('红750nm','绿540nm','蓝440nm'); grid on;
关键：sinc²(β)的写法——(sin(β)/β)²，β=πa·sinθ/λ
+1e-12：极小值防止β=0时0/0报错
规律：缝越窄→衍射越宽；波长越长→衍射越宽

第10题 光栅衍射

光学：I=I₀[sinβ/β]²·[sin(Nα)/sinα]²。核心：主极大、次极大、缺级。
% ====== 第10题：光栅衍射 ======
clear; close all;
lambda = 500e-9;  L = 1;
a = 2e-6;  b = 6e-6;  d = a + b;               % 缝宽a,挡板b,光栅常数d。d/a=4

figure('Name','第10题');
for i = 1:4
    N = N_vals(i);
    beta = pi * a * sin(theta) / lambda;        % 单缝衍射因子 β
    alpha_g = pi * d * sin(theta) / lambda;     % 多缝干涉因子 α
    
    % 光栅总光强 = 单缝衍射包络 × 多缝干涉条纹
    I_single = (sin(beta+1e-12)./(beta+1e-12)).^2;          % 衍射部分
    I_multi  = (sin(N*alpha_g+1e-12)./(sin(alpha_g+1e-12))).^2; % 干涉部分

    subplot(2,2,i);
    plot(y*1000, I); xlabel('y(mm)'); ylabel('I');
    title(sprintf('N=%d缝', N)); grid on;
end
fprintf('d/a=%.0f → 第%.0f,%.0f,%.0f…级缺级\n', d/a, d/a, 2*d/a, 3*d/a);

第12题 半圆环安培力

电磁：dF=I·dl×B，用integral数值积分求总力。
% ====== 第12题：通电半圆环所受安培力 ======
clear; close all;
mu0 = 4*pi*1e-7;  I1 = 1;  I2 = 0.5;  a = 0.1;

d_list = linspace(1.05*a, 5*a, 200);            % 距离从略大于a到5a
F_total = zeros(size(d_list));                  % 预分配存力

for idx = 1:length(d_list)
    d = d_list(idx);
    
    % 半圆环部分：对角度φ从0到π积分
    % 被积函数是 dF/dφ = I₂·μ₀I₁/(2π)·a·sinφ/(d+a·cosφ)
    f_half = @(phi) I2*mu0*I1/(2*pi) * a*sin(phi) ./ (d + a*cos(phi));
    F_half = integral(f_half, 0, pi);           % integral做数值定积分
    
    % 直径部分：对x从d-a到d+a积分
    f_diam = @(x) I2*mu0*I1./(2*pi*x);
    F_diam = integral(f_diam, d-a, d+a);
    
    F_total(idx) = F_half + F_diam;             % 两部分力相加
end

figure('Name','第12题');
plot(d_list/a, F_total*1e6);                    % ×1e6转μN
xlabel('d/a'); ylabel('F(\muN)'); grid on;
title('安培力与距离关系');
integral：做数值定积分，用法 integral(@被积函数, 下限, 上限)
安培力：dF=I·dl×B，矢量叉积的方向用右手定则判断

第13题 旋转线圈电动势

电磁：ε=-dΦ/dt。先算磁通量Φ(t)，再数值求导得电动势。
% ====== 第13题：旋转线圈感应电动势 ======
clear; close all;
mu0 = 4*pi*1e-7;  I = 1;
a = 0.05;  b = 0.1;                              % 线圈半宽a=5cm，长b=10cm
omega = 2*pi*10;                                 % 角频率 10Hz → ω=20π rad/s

d_list = [0.1, 0.2, 0.5];                        % 三种距离
t = linspace(0, 0.2, 2000);                      % 时间0~0.2s

figure('Name','第13题');
for di = 1:3
    d = d_list(di);
    epsilon = zeros(size(t));                    % 预分配存电动势
    Phi_prev = 0;                                % 上一时刻的磁通量
    
    for j = 1:length(t)
        ct = cos(omega * t(j));                  % cos(ωt)，线圈旋转角度
        % 磁通量Φ = ∫₋ₐᵃ μ₀I·b/(2π(d+x·cosωt)) dx
        f_flux = @(x) 1 ./ max(d + x*ct, d*0.001);% max防止分母变0
        Phi = mu0*I*b/(2*pi) * integral(f_flux, -a, a);
        
        if j > 1
            % 数值求导：ε = -dΦ/dt ≈ -(Φ_new - Φ_old)/Δt
            epsilon(j) = -(Phi - Phi_prev)/(t(j)-t(j-1));
        end
        Phi_prev = Phi;
    end
    
    subplot(3,1,di);                             % 3行1列，第di个
    plot(t*1000, epsilon*1000);                   % ms和mV单位
    xlabel('t(ms)'); ylabel('ε(mV)');
    title(sprintf('d=%.2fm', d)); grid on;
end
法拉第定律：ε=-dΦ/dt。先integral算Φ，再通过前后时刻差值求ε
max(a,b)：取a和b中较大的值，防分母趋零

% ====== 第14题：一维无限深势阱 ======
clear; close all;
L = 1e-9;                                        % 阱宽 1nm = 1e-9 m
h_bar = 1.054571817e-34;                         % 约化普朗克常数 ħ (J·s)

    % 能级公式
    E_n = n^2 * pi^2 * h_bar^2 / (2*m_e*L^2);   % n²成正比

    psi = sqrt(2/L) * sin(n*pi*x/L);             % 驻波形式

% ====== 第15题：康普顿散射 ======

xlabel('θ(°)'); ylabel('λ(pm)'); title('散射波长'); grid on;
subplot(2,2,4); plot(theta*180/pi, p_e);

fprintf('λ_C=%.3fpm, θ=180°时Δλ_max=%.3fpm\n', lambda_C*1e12, 2*lambda_C*1e12);

第18题 船的运动（非线性阻力）

物理：m·dv/dt=-k·vⁿ。n不同，衰减方式完全不同。
% ====== 第18题：船的运动——非线性阻力 ======
clear; close all;
m = 1000;  v0 = 10;  k = 50;                     % 质量、初速、阻力系数
t = linspace(0, 100, 2000);                      % 时间
n_vals = [0, 0.5, 1, 1.5, 2];                    % 五种n值

figure('Name','第18题');
for i = 1:5
    n = n_vals(i);
    
    if n == 0                                    % n=0:匀减速，有限时间v→0
        t_stop = m*v0/k;
        v = max(v0 - k/m*t, 0);                 % max(…,0)确保速度不为负
    elseif n == 1                                % n=1:指数衰减 e^{-kt/m}
        v = v0 * exp(-k/m*t);
    else                                        % n≠0,1:幂律衰减 v(t)=[v0^{1-n}+(n-1)kt/m]^{1/(1-n)}
        v = (v0^(1-n) + (n-1)*k/m*t).^(1/(1-n));
        if n < 1                                    % n<1时有限时间停止
            tau = v0^(1-n)/((1-n)*k/m);          % 停止时间
            v(t > tau) = 0;                      % 停止后速度为0
        end
    end
    
    plot(t, v); hold on;                         % 叠加画5条曲线
end
xlabel('t(s)'); ylabel('v(m/s)');
legend('n=0','n=0.5','n=1','n=1.5','n=2'); grid on;
n=0匀减速→有限时停止；n=1指数→永不停但递减快；n=2→初段减速快尾段极慢

第19题 单摆冲击

物理：能量守恒½mv₀²=½ml²θ'²+mgl(1-cosθ)。绳张力T=mgcosθ+mlθ'²。
% ====== 第19题：单摆冲击——角速度与张力 ======
clear; close all;
l = 1;  g = 9.8;  m = 1;                         % 摆长、重力加速度、质量
theta = linspace(0, pi, 500);                    % 角度范围
v0_vals = [2, 4, 6, 8];                          % 四种初速度

figure('Name','第19题');
for i = 1:4
    v0 = v0_vals(i);
    % 由能量守恒得：θ'² = v₀²/l² - 2g(1-cosθ)/l
    omega2 = v0^2/l^2 - 2*g*(1-cos(theta))/l;
    valid = omega2 >= 0;                         % 筛选可达角度（ω²≥0才物理）
    omega = sqrt(omega2(valid));                 % 角速度（只取有效部分）
    theta_v = theta(valid);                      % 对应的有效角度
    
    T = m*g*cos(theta_v) + m*l*omega.^2;         % 绳张力（向心力+重力分量）

    ylabel('T(N)');
    xlabel('θ(°)'); title(sprintf('v0=%.0f', v0)); grid on;

第20题 微观粒子波动性

量子：λ=h/p，非相对论λ=h/√(2meU)，相对论λ=hc/√[eU(2mc²+eU)]。
% ====== 第20题：德布罗意波 + 电子双缝干涉模拟 ======
clear; close all;
h = 6.626e-34;  c = 2.998e8;  m_e = 9.109e-31;  e = 1.602e-19;

% ---- (a) 德布罗意波长 ----
U = logspace(1, 5, 500);                         % 电压范围10V~100kV，对数均匀分布
lam_nr = h ./ sqrt(2*m_e*e*U);                   % 非相对论波长
lam_r  = h*c ./ sqrt(e*U.*(2*m_e*c^2+e*U));      % 含相对论修正波长

subplot(2,2,1);
loglog(U/1000, lam_nr*1e12, 'b-', U/1000, lam_r*1e12, 'r--');
% loglog：x和y轴都用对数坐标
xlabel('U(kV)'); ylabel('λ(pm)');
legend('非相对论','相对论'); grid on;

% ---- (b) 电子双缝干涉蒙特卡罗模拟 ----
N = 5000;                                        % 模拟5000个电子
d = 1e-6;  a = 0.1e-6;  L_screen = 1;           % 缝距、缝宽、屏距
lam_e = 0.005e-9;                                % 电子波长5pm

y_bins = linspace(-5e-3, 5e-3, 100);             % 屏上100个统计区间
I_counts = zeros(size(y_bins));                  % 每个区间的电子计数

for k = 1:N
    % 双缝干涉的概率密度（cos²调制 × sinc²包络）
    prob = cos(pi*d*y_bins/(lam_e*L_screen)).^2 .* ...
           (sin(pi*a*y_bins/(lam_e*L_screen))./(pi*a*y_bins/(lam_e*L_screen)+1e-12)).^2;
    prob = prob / sum(prob);                     % 归一化，使总概率=1
    
    % randsample：按给定的概率分布随机抽取一个区间编号
    idx = randsample(length(y_bins), 1, true, prob);
    I_counts(idx) = I_counts(idx) + 1;           % 该区间计数+1
end

subplot(2,2,[3 4]);
bar(y_bins*1000, I_counts);                      % bar画柱状图（直方图）
xlabel('屏上位置(mm)'); ylabel('电子计数');
title(sprintf('电子双缝干涉 N=%d', N)); grid on;

fprintf('U=100V:λ=%.1fpm  U=10kV:λ=%.1fpm  U=100kV:λ=%.1fpm\n', ...
    h/sqrt(2*m_e*e*100)*1e12, h/sqrt(2*m_e*e*10000)*1e12, ...
    h*c/sqrt(e*1e5*(2*m_e*c^2+e*1e5))*1e12);
loglog：双对数坐标，适合展示幂律关系
randsample：按概率权重随机抽样，模拟量子随机性
U<20kV用非相对论，U>20kV必须加相对论修正
